S2FM Audio Extractor
一个为 Source 2 Filmmaker (S2FM) 设计的高级音频提取与时间轴重建工具。它可以将 DMX 工程中的音频片段导出为标准的 XML 格式，保留原始时间轴位置，并利用智能算法进行分轨优化。

✨ 核心特性

智能分轨 (Smart Tracking)：

基于音频前缀、家族名和文件目录自动分轨。

紧凑布局：自动处理重叠片段，将相似音频归类到同一轨道，最大化节省纵向空间。

精准时间轴：支持自定义 FPS（默认 60），支持负时间轴平移。

深度搜索匹配：

即便 DMX 中只有虚构的 vsnd 路径，工具也能在你的本地音频库中通过相对路径、文件名、词干 (Stem) 等多种维度自动找回原始 .wav 或 .mp3 文件。

轨道过滤：支持对 DMX 中的轨道进行搜索、筛选、全选或反选，仅导出你关心的部分。

🚀 快速开始

dmxconvert.exe：通常位于 Counter-Strike 2\game\bin\win64 目录下。

音频库：从CS2游戏提取出的全部音频文件夹。

✨ 使用方法：

选择 dmxconvert.exe 的路径。

选择你的 .dmx 工程文件。

选择你的音频资源根目录。

读取轨道：点击“读取 DMX 声音轨道”解析工程。

生成 XML：在右侧列表勾选需要导出的轨道，点击“生成 XML”。
